## 🔍 Проект 1: Logo-Skan-AI
Приложение для определения оригинальности логотипов с использованием технологий машинного зрения. Помогает проверить, не скопирован ли бренд.

### ⚙️ Как это работает:
1. **Загрузка:** Пользователь загружает изображение логотипа в систему.
2. **Анализ (Хэширование):** Алгоритм обрабатывает картинку и вычисляет её уникальный цифровой отпечаток (хэш).
3. **Поиск совпадений:** Происходит мгновенное сравнение полученного хэша с нашей базой данных существующих брендов.
4. **Вердикт:** Система выдает результат о степени уникальности логотипа.

## 🎙️ Проект 2: Голосовое управление AI (Scratch + ML4K)
Интерактивная игра, в которой главный персонаж (спрайт) понимает человеческую речь и меняет свое поведение в зависимости от того, что ему говорят.

### 🧠 Как работает машинное обучение (ML4K):
В этом проекте я использовал платформу **Machine Learning for Kids**. 
1. **Сбор данных:** Я создал две категории (корзины) для фраз — позитивные (например, "ты молодец", "отличная работа") и негативные ("плохо", "ужасно").
2. **Тренировка модели:** Искусственный интеллект проанализировал эти примеры и научился понимать эмоциональную окраску текста (Sentiment Analysis).
3. **Интеграция в Scratch:** Модель подключена к Scratch. Когда игрок произносит фразу в микрофон, ML4K распознает текст, классифицирует его эмоцию, и спрайт реагирует соответствующим образом (радуется или грустит).

## 🚀 Инструкция по запуску

### Для Logo-Skan-AI:
1. Перейдите по ссылке на [веб-приложение](https://d68856815-beep.github.io/Logo-skan-AI-/).
2. Нажмите кнопку загрузки и выберите файл с логотипом (форматы `.png` или `.jpg`).
3. Дождитесь ответа от нашего бэкенд-сервера с результатами проверки.

### Для AI-Игры на Scratch:
1. Откройте проект в среде Scratch (с подключенным расширением ML4K).
2. Разрешите браузеру доступ к микрофону.
3. Нажмите на Зеленый флажок.
4. Произносите в микрофон разные фразы и наблюдайте за реакцией ИИ!
