# Logo-skan-AI
Проект по определению оригинальности логотипов с помощью технологий машинного зрения.

## Как это работает?
1. Пользователь загружает изображение.
2. Алгоритм вычисляет хэш изображения.
3. Происходит сравнение с базой данных.
